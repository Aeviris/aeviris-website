# Aeviris Homepage (Vite + React + Tailwind)

Quick scaffold for a premium homepage design for Aeviris.

Getting started:

```powershell
npm install
npm run dev
```

This project includes:
- Dark theme, glassmorphism, and neon gradients (Tailwind)
- Framer Motion for subtle animations
- Sections: Hero, What We Build, Current Product, Why Aeviris, Vision & Mission, Roadmap, CTA, Footer
